How to deal with text as data

Mozilla Festival 2013

Download .zip Download .tar.gz View on GitHub

This repository contains the slides and code for the workshop on working with text as data for the Mozilla Festival 2013 in London. If you have never cloned a git repository before, follow the bootcamp provided by GitHub: https://help.github.com/categories/54/articles

$ git clone https://github.com/rjweiss/mozfest2013.git
$ cd mozfest-2013-slides
$ bash serve.sh 
# this will open up a browser and start serving the slide show

Authors and Contributors

Rebecca Weiss (@rjweiss) for the Mozilla Festival 2013

Additional information

  • Slides are HTML5 slides from Google: https://code.google.com/p/io-2013-slides/
  • R scripts are distributed as R Markdown -- Unfortunately, there isn't a nice site like nbviewer.ipython.org...yet
  • Python scripts are distributed IPython notebooks -- If you can't run a Python IDE that can handle notebooks, try going to these links: -- Python classification gist -- Python sentiment analysis gist

Recommended resources for the workshop

IDEs:

  • RStudio (for R): http://www.rstudio.com/
  • Canopy (for Python): https://www.enthought.com/products/epd/free/ -- also includes many useful Python dependencies
  • IPython: http://ipython.org/install.html
