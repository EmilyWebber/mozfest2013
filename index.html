This project is maintained by rjweiss

This repository contains the slides and code for the workshop on working with text as data for the Mozilla Festival 2013 in London. If you have never cloned a git repository before, follow the bootcamp provided by GitHub: https://help.github.com/categories/54/articles

$ git clone https://github.com/rjweiss/mozfest2013.git
$ cd mozfest-2013-slides
$ bash serve.sh 
# this will open up a browser and start serving the slide show

If you don't want to bother with downloading and running the code yourself, you can see the Python notebooks as static webpages here:

  • Simple text classification
  • Simple sentiment analysis
