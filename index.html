This repository contains the slides and code for the workshop on working with text as data for the Mozilla Festival 2013 in London. If you have never cloned a git repository before, follow the instructions below

$ # Move to a directory you have write access to
$ git clone https://github.com/rjweiss/mozfest2013.git
$ cd mozfest-2013-slides
$ bash serve.sh 
# this will open up a browser and start serving the slide show
